# Kiosk User Guide

## Purpose

This kiosk collects patient check-in details and (in future) vital signs. Today, the vital measurement step is a **simulation** for prototype purposes.

## Typical flow

1. Tap **Start Check-In**
2. Enter **Personal Information**
   - First name, last name
   - Date of birth
   - Gender and civil status
3. Enter **Contact & Guardian Details**
   - Phone, address
   - Guardian info (optional)
4. Follow the **Vital Signs** steps
   - The kiosk will “measure” vitals automatically (prototype simulation)
5. **Review** your information
   - Confirm checkbox
   - Tap **Submit**
6. **Success**
   - Reference ID displayed
   - System returns to home after countdown

## Assistance

- Tap the “Request Help” option during vital measurement steps.
- The system shows “Assistance Requested” and displays a kiosk location label (currently hardcoded).

## Accessibility

The toolbar supports:
- Font size changes
- High contrast mode
- Audio / screen reader options (basic)

## Important notes

- Do not enter highly sensitive information beyond what is requested.
- This is a prototype: vital sign values shown are simulated.
